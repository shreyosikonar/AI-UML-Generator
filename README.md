# AI-Based Automated UML Diagram Generator

An AI-powered system that converts Software Requirement Specification (SRS) documents into UML components automatically using NLP.

## Features
- SRS preprocessing
- UML component extraction
- Relationship classification
- Confidence scoring
- Human-in-the-loop correction

## Tech Stack
- Python
- FastAPI
- spaCy
- Scikit-learn
- HTML/CSS

## How to Run

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
uvicorn app:app --reload
